# KC UI

**Prerequisites:**

* install node & npm
* install yarn
* install and configure the aws cli

**Development:**

* clone the repo
* `cd` into it
* `yarn` to install deps
* `VARIATION=variation-name yarn start` to launch the dev server

**Deployment:**

* `VARIATION=variation-name yarn run deploy:demo` deploys to https://kc.demo.spraoi.ai
